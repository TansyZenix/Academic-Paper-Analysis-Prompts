# Academic Paper Analysis Prompts

> 面向计算机视觉 / AI 论文的**结构化分析提示词库**
>
> A structured prompt library for in-depth CV/AI paper analysis

---

## 简介

本项目提供一套完整的、结构化的 Prompt 提示词模板，用于系统性地分析和理解计算机视觉 / AI 领域学术论文。适用于科研入门者、论文复现者以及对论文深读有需求的研究人员。

每个模块针对论文的不同部分，设计了精细的指令约束，确保输出结果具备**可复现性**和**完整性**。

## 功能模块

| 模块 | 功能 | 适用论文部分 |
|------|------|-------------|
| 1. 文献筛选与推荐 | 按 4090 可复现性筛选高价值论文 | 文献调研 |
| 2. 领域理解与导读 | 初学者友好的论文核心解读 | 整体理解 |
| 3. 引言深入解析 | 从背景到创新的逻辑链分析 | Introduction |
| 4. 相关工作结构化解析 | 逐个方法带 shape 数据流拆解 | Related Work |
| 5. 任务形式化解析 | 输入→中间过程→输出的完整推导 | Task Formulation |
| 6. 模型框架深度解析 | 模块级可复现解析 + 实现提示 | Framework/Method |
| 7. 评价指标 | 指标公式 + 数值计算示例 | Experiments |
| 8. 实验 | 超参数、设置、可复现性分析 | Experiments |
| 9. 数据集 | 数据内容、预处理、划分方式 | Datasets |
| 10. 损失函数 | 总损失→子损失→数值示例逐层拆解 | Loss Function |

### 模块特点

- **统一的 Shape 规范**：全程使用 $\mathbb{R}^{B \times H \times W \times C}$ 统一表示
- **强制完整性检查**：每个模块都有防遗漏机制
- **数值示例**：关键模块包含可手动计算的最小化示例
- **复现导向**：明确标注"合理推测"与"未提及"
- **可复现级解析**：内容清晰到可直接画结构图或写 PyTorch 代码

## 使用方式

1. 将论文 PDF / Markdown 内容提供给 AI 助手
2. 选择对应论文部分的 Prompt 模块
3. AI 将按照模块中定义的严格格式输出结构化分析

```
例如：
- 想理解模型结构 → 使用模块 6（模型框架深度解析）
- 想复现实验 → 使用模块 8（实验）+ 模块 10（损失函数）
- 筛选可复现论文 → 使用模块 1（文献筛选与推荐）
```

## 依赖

- 支持 Prompt 的 AI 助手（如 Claude、GPT 等）
- 用于渲染数学公式的 Markdown 渲染器（推荐 VSCode + Markdown Preview Enhanced）
